Virtual Fortress Multi Scan









For multi scan:
1) Enter the range of IP addresses of the target machines whose hardening levels need to be checked to ensure security of systems.
2) Start the scan after providing the respective IP addresses to the tool.
3) Open the shell to enter the encryption keys which ensure end-to-end privacy between all stations.
4) Executing the commands in the shells of the target systems takes time. This includes complete and secure execution of all said shell commands.
5) The report of the findings of VirtualFortress is displayed in the form of an HTML report which helps in visualizing what went wrong. This report can be found in the source folder after completion of the scan
